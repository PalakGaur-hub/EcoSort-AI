# EcoSort-AI

An AI-powered waste segregation system that classifies images into biodegradable and non-biodegradable categories using a CNN model. The system includes a confidence-based prediction mechanism and user-assisted correction for improved accuracy in real-world scenarios.

## Features

* Image-based waste classification
* Confidence-based prediction system
* User-assisted correction for uncertain cases
* Interactive frontend interface

## Technologies Used

* Python
* TensorFlow / Keras
* HTML, CSS, JavaScript
* NumPy, PIL

## How it Works

1. User uploads an image
2. Model predicts waste category
3. Confidence score is generated
4. If uncertain, user input is requested
5. Final classification is displayed

## Applications

* Smart waste management
* Recycling systems
* Environmental awareness
